# Todo List

## Описание проекта

`Todo List` — это простое React-приложение для управления списком задач.

Frontend-проект демонстрирует базовую работу с компонентами React, хуками, состоянием приложения и сохранением данных в `localStorage`.

## Функциональность

Приложение позволяет:

- добавлять новые задачи;
- отмечать задачи как выполненные;
- удалять задачи;
- редактировать текст задачи;
- фильтровать список задач:
  - все задачи;
  - невыполненные;
  - выполненные;
- сохранять задачи между перезагрузками страницы через `localStorage`.

## Как запустить проект

### 1. Клонировать репозиторий

```bash
git clone https://github.com/AlexFeed/todo_list.git
cd todo_list
```

### 2. Установить зависимости

```bash
npm install
```

### 3. Запустить проект

```bash
npm start
```

После запуска приложение будет доступно по адресу:

```text
http://localhost:3000
```

## Доступные команды

```bash
npm start
```

Запускает проект в режиме разработки.

```bash
npm run build
```

Собирает production-версию приложения.

```bash
npm run deploy
```

Публикует приложение на GitHub Pages.
